## 简介

vue-element-admin) 是一个后台前端解决方案，它基于Vue 和 element-ui实现。
它使用了最新的前端技术栈，内置了 i18n 国际化解决方案，动态路由，权限验证，提炼了典型的业务模型，提供了丰富的功能组件，它可以帮助你快速搭建企业级中后台产品原型。
相信不管你的需求是什么，本项目都能帮助到你。
